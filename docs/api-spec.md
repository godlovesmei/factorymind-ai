# API Specification

Milestone 1 exposes health checks only:

```txt
GET /health
GET /api/health
```

Planned API groups from `PLAN.md`:

- `/api/auth`
- `/api/machines`
- `/api/sensor-logs`
- `/api/alerts`
- `/api/reports`
- `/api/ai`
- `/api/maintenance`

Detailed request and response contracts will be added as each module is implemented.
