# Architecture

FactoryMind AI is planned as a modular full-stack manufacturing operations system.

Current milestone status:

- `apps/web`: Next.js frontend foundation.
- `apps/api`: Node.js/Express API foundation.
- `packages/shared`: Shared TypeScript contracts.
- `packages/database`: Prisma and MySQL foundation.

Future milestones will add MQTT ingestion, Socket.IO realtime updates, database models, AI report generation, and legacy maintenance integration.
