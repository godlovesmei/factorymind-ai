# Deployment

Deployment is not part of Milestone 1.

Planned deployment targets:

- Frontend: Vercel
- Backend: Render, Railway, Fly.io, or local demo
- Database: TiDB Cloud Starter or local MySQL
- MQTT: HiveMQ Cloud or local Mosquitto
- ML service: Hugging Face Spaces or local fallback

Deployment commands and environment setup will be documented after the core app is running locally.
